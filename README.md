# FrameworkVision Engineering 🏗️

Plataforma SaaS inteligente para Engenharia, Arquitetura e Construção.

## 🚀 Características Principais

- ✅ Autenticação com Email/Password e Google OAuth
- ✅ Dashboard completo para usuários
- ✅ Gestão de projetos
- ✅ Cálculos de engenharia com IA
- ✅ Orçamentação automática
- ✅ Marketplace de profissionais
- ✅ Sistema de notificações
- ✅ Design premium responsivo

## 📋 Requisitos

- Node.js 18+
- npm ou yarn

## 🛠️ Instalação

```bash
npm install
npm run dev
```
